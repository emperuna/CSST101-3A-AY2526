Mini Expert System

Python Script PDF Documentation September 2025

Overview

A university logic rules system implementing propositional logic with implication operators (P → Q). The system evaluates various academic rules using logical conditions and produces human-readable results.

Key Features

  • Attendance rule checker (late + excuse letter)
  • Grading rule evaluation (pass/fail logic)
  • Login system simulation
  • Bonus points eligibility checker
  • Library borrowing rule verification
  • CSV logging for all results

AI Concepts

Propositional Logic Implication (P → Q) Rule-Based Systems Expert Systems

Code Snippet

mini-expert-system.py 
# Logic function - Implication (P → Q)
def impl(P, Q):
    return (not P) or Q

# Attendance Rule Checker
def attendance_rule(student_name):
    late = input("Was the student late? (T/F): ")
    excuse = input("Did the student provide an excuse? (T/F): ")
    
    result = impl(late, excuse)
    outcome = "Satisfied" if result else "Violated"
    print(f"Result: {outcome}")
