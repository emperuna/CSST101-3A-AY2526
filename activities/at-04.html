Non-Monotonic Reasoning

Python Script • Draw.io Diagram • PDF Documentation

Overview

A bird reasoner demonstrating non-monotonic logic — where new information can invalidate previous conclusions. The system applies default reasoning (birds can fly) then revises beliefs when exceptions are discovered.

Key Features

  • Default rule: "If X is a bird, assume X can fly"
  • Exception handling for flightless birds
  • Known exceptions: penguin, ostrich, kiwi, emu, cassowary, dodo
  • Step-by-step reasoning trace output
