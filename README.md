## Overview

This repository contains coursework, exercises, and projects for **CSST 101: Advanced Representation and Reasoning in AI**. The portfolio demonstrates practical implementations of knowledge representation, logical reasoning, and intelligent systems.

### Learning Objectives

- Design and implement rule-based expert systems
- Apply propositional and predicate logic for knowledge representation
- Implement inference mechanisms (forward/backward chaining)
- Understand non-monotonic reasoning and belief revision

---

## Coursework

| Activity  | Title                                                       | Description                                                        | Key Concepts                           |
| --------- | ----------------------------------------------------------- | ------------------------------------------------------------------ | -------------------------------------- |
| **MP-01** | [Mini Expert System](coursework/machine-problem-01/)        | University logic rules system implementing propositional logic     | Expert Systems, Implication (P → Q)    |
| **MP-02** | [Predicate Logic](coursework/machine-problem-02/)           | Implementation of unification and inference algorithms             | Unification, Forward/Backward Chaining |
| **MP-03** | [Troubleshooting Assistant](coursework/machine-problem-03/) | Rule-based expert system for computer diagnosis                    | Rule-Based Systems, Decision Trees     |
| **AT-04** | [Non-Monotonic Reasoning](coursework/assessment-task-04/)   | Bird reasoner demonstrating belief revision and exception handling | Default Logic, Belief Revision         |

## Getting Started

### Prerequisites

```bash
pip install jupyter
```

### Running Notebooks

1. Clone the repository:

   ```bash
   git clone https://github.com/emperuna/CSST101-3A-AY2526.git
   cd CSST101-3A-AY2526
   ```

2. Launch Jupyter:

   ```bash
   jupyter notebook
   ```

3. Navigate to `coursework/` and open any `.ipynb` file

### Viewing the Portfolio

Open `index.html` in your browser to view the web portfolio.
